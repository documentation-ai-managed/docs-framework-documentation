## Advanced Features

<Expandable title="Template Variables" default-open="false">
  Use variables to create dynamic content that adapts to different contexts.
  
  ```javascript
// Example template with variables
const template = {
  content: 'Welcome to {store_name}! Check out our {product_category} specials.',
  variables: {
    store_name: 'Downtown Location',
    product_category: 'coffee'
  }
};
```
</Expandable>
